🚀 MZANSI CHRONICLES

📌 Description
A south African based platformer that treats users to an over exaggerated "experience" of the genuine Mzansi experience taking users across the treasured cities Joburg, cpt, and Kzn.

✨ Features
- **3 Distinct City Levels:** Journey through uniquely styled levels for Johannesburg, Cape Town, and KwaZulu-Natal.
- **Authentic Mzansi Feel:** Over-exaggerated South African cultural elements, obstacles, and references.
- **Fast-Paced Mechanics:** Smooth platforming, jumping, and dodging built entirely with Phaser.js.

🛠️ Technologies Used
- Phaser.js
- HTML5
- CSS3
- JavaScript
- Vite (Bundler & Dev Server)

▶️ How to Run
1. Clone or download the project repository.
2. Ensure you have [Node.js](https://nodejs.org/) installed.
3. Open your terminal in the project directory and run:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
5. Open the local URL provided by Vite in your browser to play!

*(Alternatively, if a production build is provided, you can serve the `dist/` folder.)*

🎮 Controls
- **Movement:** `Arrow Keys` or `W`, `A`, `S`, `D` to navigate.
- **Action:** `Spacebar` to jump / interact.

- **Learning Scene Management:** Understanding how to seamlessly transition between different city states and menus in Phaser.js.
